# learnflow-app

LearnFlow is an AI-powered Python tutoring platform that uses a microservices architecture with      
  specialized agents (concepts, debug, code-review, exercise, progress) to provide personalized        
  learning through an interactive chat tutor, Monaco-powered code editor with live execution, adaptive 
  quizzes, and mastery-based progress tracking. It features a dark red/white/black themed UI with GSAP 
  animations, a Next.js frontend, and a FastAPI backend with an intelligent triage agent that routes   
  student queries to the appropriate specialist agent.
